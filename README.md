## Tech Stack

- Next

## How to run project

### 1. Clone the project

```bash
git clone https://github.com/nguyenhuybinh270709/JUNIOR-CEO.git
cd JUNIOR-CEO
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run project

```bash
npm run dev
```
